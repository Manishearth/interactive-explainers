🧩 Polynomial Roots as Swap Targets

Every polynomial equation of degree $n$ has $n$ roots ($r_1, r_2, \dots, r_n$) in the complex plane. The Symmetric Group $S_n$ consists of all possible ways to shuffle (permute) these roots.

Vieta's Formulas (Coefficient Invariance):

$P(x) = (x - r_1)(x - r_2)\dots(x - r_n) = x^n - (r_1 + \dots + r_n)x^{n-1} + \dots$

Shuffling roots changes the order of solutions, but leaves the equation's coefficients completely unchanged!

Group Size |S_n| = n! 6 elements

S₃ has 3! = 6 permutations: 3 reflections (swaps) and 3 rotations.

🔄 What is an "Abelian Chunk"? (Commutativity vs Scrambling)

An Abelian group is a group where operations are commutative: performing Action A then Action B produces the exact same result as performing Action B then Action A ($A \circ B = B \circ A$).

1. Clock Turns (Abelian / Commutative)

A ∘ B = B ∘ A

Turning a clock hand by +3 hours then +5 hours yields +8 hours (or 8 o'clock). Doing +5 hours then +3 hours gives the exact same 8 o'clock!

A then B +3h → +5h = +8h
=
B then A +5h → +3h = +8h

Radical signs ($\sqrt[n]{\cdot}$) strictly generate abelian cyclic groups ($\mathbb{Z}_n$). They are "clock turn" operations!

2. Root Swaps (Non-Abelian / Order Matters)

A ∘ B ≠ B ∘ A

Try swapping roots! Action A: Swap (r₁, r₂). Action B: Swap (r₂, r₃). Watch how doing A then B produces a completely different root arrangement than B then A!

When group elements tangle, we need to test their commutator $[A, B] = A B A^{-1} B^{-1}$.

🏰 The Solvable Factorization Tower ($S_2 \to S_5$)

Galois proved that a polynomial equation can be solved by radicals if and only if its symmetry group can be sliced down to $\{1\}$ using only Abelian factor groups ($\mathbb{Z}_p$).

Why A₅ is an Unbreakable Brick Wall:

$A_5$ has size 60 and is a SIMPLE GROUP (it contains no non-trivial normal subgroups). Furthermore, $A_5$ is non-abelian. Therefore, you cannot slice $A_5$ any further into abelian chunks!

🔑 The Galois Gate: Radical Keys vs Symmetry Locks

Why does group solvability dictate whether a polynomial formula exists? Taking an $n$-th root ($\sqrt[n]{a}$) is algebraically equivalent to creating a Cyclic Extension Field ($\mathbb{Z}_n$). Radical signs are "Abelian Keys".
